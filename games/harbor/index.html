How to Play
  • 1Tap each gray vehicle to set move order (1, 2, 3…). Tap again to undo from that point. The pink car is not selectable.
  • 2Tap the arrow on a selected vehicle to flip slide direction. Each vehicle slides as far as it can that way.
  • 3Wrong plan rewinds the jam. Clear a path so the pink car can roll through the exit hole.
